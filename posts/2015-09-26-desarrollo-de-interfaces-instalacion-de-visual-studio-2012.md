En el módulo de Desarrollo de interfaces nos ocuparemos de... ¡las interfaces! Que poder de deducción tenéis, jodios. Bueno, esta aplicación la usaremos para programar en Visual Basic, lenguaje que aprendí a los doce años y lo dejé a los trece (descubrí el mundo de las redes y los sistemas y me engancharon muchísimo más).

Como fuere, para la instalación ejecutamos el programa, aceptamos los términos y condiciones y, si se desea, podemos unirnos al programa para la mejora de la experiencia del usuario.

![Pantalla de inicio del instalador](/img/posts/20150926_40.png)

## Selección de características

Lo siguiente será seleccionar las características a instalar. Desmarcamos las siguientes opciones:

- **Blend para Visual Studio**: se utilizaría para programar en Windows 10 con soporte Touch.
- **LightSwitch**: herramienta para crear aplicaciones empresariales de forma rápida.

Desmarcadas ya, pulsamos sobre **Instalar**.

![Selección de características a instalar](/img/posts/20150926_41.png)

## Descarga e instalación

Y bueno, tenemos paciencia. Comenzará la descarga y aplicación de los módulos correspondientes.

![Descarga e instalación de módulos en progreso](/img/posts/20150926_42.png)

Cuando haya finalizado, podremos iniciar el programa.

![Instalación completada](/img/posts/20150926_43.png)

## Activación

Deberemos introducir el número de serie para activar la aplicación y pulsamos en **Siguiente**.

![Pantalla de introducción del número de serie](/img/posts/20150926_44.png)

## Configuración del entorno

Realizada la activación, se nos pedirá que elijamos la configuración de entorno predeterminada. Seleccionamos **Visual Basic**, que es como vamos a desarrollar. Si en el futuro queremos cambiarla, podemos hacerlo desde **Herramientas → Importar y exportar configuraciones → Restablecer todas las configuraciones**.

![Selección de configuración de entorno predeterminada](/img/posts/20150926_45.png)

![Menú de configuración de Visual Studio](/img/posts/20150926_46.png)

## Pantalla inicial

Y esta será la pantalla inicial, desde la que podremos elegir cómo programar o incluso ver vídeos para aprender cómo iniciarnos con el entorno de Visual Studio 2012.

![Pantalla de inicio de Visual Studio 2012](/img/posts/20150926_47.png)

Cómo se programa lo veremos en futuras entregas.
