## ¿Cómo usar MySQL WorkBench? Primeros pasos

(Notas muy rápidas para saber desenvolvernos)

**New Model**

![New Model en MySQL WorkBench](/img/posts/20141014_1.png)

**New Diagram**

![New Diagram en MySQL WorkBench](/img/posts/20141014_2.png)

Aparece este cuadrado

![Cuadrado de diagrama](/img/posts/20141014_3.png)

Este es el menú de opciones ampliado

![Menú de opciones](/img/posts/20141014_4.png)

Elegimos este icono

![Icono de nueva tabla](/img/posts/20141014_5.png)

para crear una nueva relación. Le damos doble clic para configurar su nombre y campos.

Table name → Nombre Entidad

Column name → Campos

![Configuración de tabla](/img/posts/20141014_6.png)

Para realizar las relaciones, elegimos uno de estos iconos

![Iconos de relaciones](/img/posts/20141014_7.png)

Y pulsamos sobre las entidades a aplicar. Eligiendo N:M quedará así:

![Relación N:M](/img/posts/20141014_8.png)

Se creará una nueva tabla, en la que añadimos el campo Nota (ejercicio 15, recuerda).

![Nueva tabla con campo Nota](/img/posts/20141014_9.png)

Para insertar datos hacemos doble clic y vamos, en la parte inferior, a Insert

![Insertar datos](/img/posts/20141014_10.png)

Para guardar, Archive → Save mode as o vamos a:

![Opción de guardar](/img/posts/20141014_11.png)

Se abre la siguiente pantalla y elegimos las opciones que deseemos:

![Pantalla de guardado](/img/posts/20141014_12.png)

*Y eso sería todo en la práctica del otro día. Realmente es una pequeña guía para poder entender un poquito el programa. Pero la verdad, lo mejor es que vayáis probando vosotros mismos y veáis sus diversas opciones.*
